## Flujo de trabajo
1. Se trabaja en Claude Code y se generan documentos en `docs/`
2. El contenido se publica como comentario en el issue correspondiente de Linear
3. Se arroba a Martín (`@msedes@remax.com.uy`) para que le llegue notificación a su inbox
4. Martín revisa y comenta cuando puede

## Estados de issues
| Estado | Uso |
|--------|-----|
| Backlog | Creado pero no planificado |
| Todo | Planificado, pendiente de inicio |
| In Progress | En trabajo activo o en revisión |
| Testing | En validación por Martín |
| Done | Completado y validado |
| Canceled | Descartado |
