# OnMind — Tipografía

> Decisión tipográfica para producto y marketing.

**Autor:** Raphael & Claude AI
**Fecha:** 2026-04-06

---

## Decisión

**Geist Sans para todo.** Una sola familia tipográfica para app, landing, redes, emails y cualquier pieza de comunicación.

## Por qué

- Geist tiene 9 pesos (Thin a Black), suficiente rango para crear jerarquía visual sin una segunda fuente.
- Una sola fuente = consistencia automática entre producto y marketing.
- Menos decisiones tipográficas = más velocidad de ejecución.
- Es la misma estética que usan marcas SaaS modernas (Linear, Vercel, Raycast).

## Cómo usarla

| Contexto | Peso | Ejemplo |
|----------|------|---------|
| Headlines / títulos grandes | ExtraBold (800) o Black (900) | Hero de landing, título de post |
| Subtítulos | SemiBold (600) o Bold (700) | Secciones, subtítulos |
| Cuerpo de texto | Regular (400) o Medium (500) | Párrafos, descripciones |
| Texto secundario | Regular (400) | Captions, notas al pie |

## Fuente complementaria (reserva)

Si en el futuro los headlines necesitan más personalidad, la primera opción es **Plus Jakarta Sans** (Google Fonts, gratuita) — humanista, profesional con calidez, buen soporte de español.

Solo se incorporaría para títulos de marketing, no para la app.

## Fuente monoespaciada

**Geist Mono** se mantiene para código y datos técnicos dentro de la app. No se usa en marketing.

*Decisión tomada priorizando simplicidad y coherencia. Se reevalúa cuando se diseñen las primeras piezas de marketing.*
